# 🌦 Weather App

A simple weather app using *HTML, **CSS, and **JavaScript* that shows real-time weather updates for any city.

## 📌 Features

- Search weather by city name
- Shows temperature, weather condition, and more
- Simple and responsive UI

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API
